# 术后即刻阶段

**术后0到10-14天**
- 石膏固定，禁止负重

**本阶段目标**
- 尽量减少感觉障碍
- 尽量减少局部和近端肌肉萎缩

# 1.感觉康复

按摩和触摸暴露的皮肤区域，以尽量减少足部和踝部躯体感觉皮层的萎缩，可将筷子伸进石膏与足底接触面摩擦适度刺激足底，因为慢性踝关节不稳定的患者经常出现足底感觉减退。

# 2.运动康复

- 保持近端髋部力量，以降低反复踝关节扭伤的风险，并改善下肢运动控制能力。（具体方法参考第一部分预康复中髋部肌力锻炼方法）
- 通过心理练习和活动对侧踝关节（同时想象患侧肌腱收缩感受），进而减少躯体运动皮层代表区的丧失。如果可以，通过石膏的窗口对肌腱施加振动。
- 被石膏固定的足按压/推顶与足接触的石膏内侧面进行各个方向的等长收缩（前提：石膏要固定确切）（等长收缩的概念：肌肉长度不变但仍产生张力的收缩模式，适用于术后早期固定阶段，避免关节活动过度牵拉修复组织），尤其是外翻，也可以进行“缩足”内在肌锻练。“缩足” 内在肌训练，即通过主动收缩足部深层小肌肉使足弓上抬、足部纵向缩短的训练，它是足踝康复核心基础训练，见下方视频：  
    - 患者先放松足部，随后向上提拉足弓，激活足部内在肌。

- 加强对侧下肢力量训练，即锻练没做手术的对侧正常腿的肌肉力量，因石膏固定侧患肢也会从健侧力量训练中获益，增加了的7%-8%的自主力量（具体方法参考第一部分预康复中髋部肌力锻炼方法）。
- 减轻疼痛和肿胀，以尽量减少关节源性肌肉抑制所引起的无力。可以通过下列方法解决：
    - 关节深度冷敷（20分钟）（冰敷方法见预康复部分）。
    - 减轻疼痛（可以选择止痛药、非甾体抗炎药、神经肌肉电刺激和呼吸放松技术）。
    - 减轻肿胀（抬高患足高于髋部甚至高于心脏）。
